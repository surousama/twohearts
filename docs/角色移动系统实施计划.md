# 角色移动系统实施计划

## 方案：Lyra 方案（Distance Matching + Stride Warping）

## 阶段一：补齐基础设施

### 1. 启用插件
- [x] AnimationWarping（已启用）
- [x] AnimationLocomotionLibrary（已添加到 .uproject）

### 2. 为重定向动画生成 Distance Curve
- 批量选中 Chinese_Warrior/Animations 下的 Start/Stop 动画
- 右键 → Animation Modifier → 添加 `DistanceCurveModifier`
- 曲线名设为 `Distance`

### 3. 为动画添加脚步同步标记
- 对 Start/Cycle 动画添加 `SyncMarkerAnimModifier`
- 自动在脚步触地点生成同步标记

### 4. 创建 Blend Space 2D
- `BS_Walk`（Speed × Direction）—— Walk Loop 动画
- `BS_Run`（Speed × Direction）—— Run Loop 动画
- `BS_Sprint`（Speed × Direction）—— Sprint Loop 动画（仅前方）

## 阶段二：搭建新 AnimBP

### Event Graph（计算层）
- GroundSpeed：水平移动速度
- DistanceTraveled：上一帧实际移动距离
- bShouldMove：是否应该移动
- MovementDirection：移动方向角度
- PredictedStopDistance：预测刹车距离

### Anim Graph（状态机）

```
Idle → Start → Cycle → Stop → Idle
                ↑                │
                └── Pivot ───────┘
```

| 状态 | 动画节点 | 驱动方式 |
|------|---------|---------|
| Idle | Sequence Player | 直接播放 Idle_Loop |
| Start | Sequence Evaluator | Advance Time by Distance Matching |
| Cycle | Blend Space | Set Playrate to Match Speed + Stride Warping |
| Stop | Sequence Evaluator | Distance Match to Target |
| Pivot | Sequence Evaluator | Advance Time by Distance Matching |

### 过渡方式
- 所有状态过渡使用 **Inertialization Blend**（惯性混合），替代传统 Cross Fade

### 后处理
- Orientation Warping（朝向适配）
- Stride Warping（步幅适配）
- Foot Placement（地形 IK，已实现）
