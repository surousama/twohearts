# 双心印战斗系统介绍

## 项目概述

双心印是一款基于UE5开发的实时动作战斗游戏，核心玩法围绕结印系统和双人合作战斗展开。本开发计划专注于战斗系统的核心功能实现，优先开发单机版本，为后续联机功能奠定基础。

## 核心功能描述

### 1. 战斗场景的基础设定
一个常规的战斗下，玩家控制一名角色，并且具有一个人形态队友，共同面对一个BOSS
这个队友初步计划由电脑控制，后续拓展为由真实玩家控制
玩家和队友之间可以通过【合印】的操作来彼此支援

### 2. 战斗系统
玩家和队友都属于同一类角色，他们的框架是一样的：
每个角色都有一套普通攻击，分别包含3个普通攻击连段
每个角色都有一套技能系统，技能系统又叫做【结印】系统
结印系统里玩家的技能数量由玩家自定义，可以装备多个技能，也可能装备少量技能
技能繁杂而多样，有一些是主动释放的，有一些是被动技能


### 3. 结印系统

玩家身上有多种多样的【印】，玩家在游戏里通过按某个按键来【结印】
点击【结印】之后，游戏内界面会出现玩家可用的技能列表。（在点击结印前无法显示）
结印是释放技能的前置步骤。

例1：
玩家操作：【三山印】-【崩雷鸣】，意思是玩家点击了三山印，并在三山印的情况下，释放了技能崩雷鸣
例2：
玩家操作：【金刚印】-【崩雷鸣】，意思是玩家点击了金刚印，并在金刚印的情况下，释放了技能崩雷鸣

玩家可以通过不同的印作为前置条件，释放同一个技能。
每一种印都有一个自带的加成效果
每一种印下面有若干个辅助宝石插槽，每种辅助宝石都有一定的加成效果，又在相互之间有一定约束


例如三山印的自带技能效果是：水属性和雷电属性技能伤害提高20%。同时范围技能伤害提高30%。辅助宝石第一个槽位效果翻倍。
三山印的技能宝石是：雷电属性伤害提高15%---命中眩晕----感电
由于三山印本身提升第一个辅助宝石效果，因此玩家使用三山印时，实际效果是雷电伤害提高了50%  （第一个宝石15% *2 = 30%+三山印自带20% ，一共50%）。同时命中敌人时会附带眩晕和感电效果。

此时假设崩雷鸣是一个范围伤害雷属性技能，则可以收到三山印的各种加成，造成大量雷伤害，并且眩晕与感电敌人。


### 4. 合印系统

两名玩家之间可以选择合印。合印具有一定的发动条件。
使用合印后，两个玩家各自选择一个印记进行相合，合印之后，双方都有各自印记的加成效果。
例如玩家A使用三山印，玩家B使用金刚印，两个玩家合印，此时玩家A释放崩雷鸣，则既可以享受到自己的三山印的效果，又可以享受到玩家B的金刚印的效果。
