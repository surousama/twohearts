# 双心印项目定位

## 核心定位

1. 双心印是一款以 `1v1` 社交和双人合作玩法为核心的新游类型
2. 战斗层采用双人共斗 ARPG 方向
3. 技能系统核心是“结印”，双人协同核心是“合印”

## 用户与内容结构补充

1. 项目同时面向社交型用户、偏外观与情感体验用户、偏动作战斗用户
2. 剧本、多结局、副本、双人关系和战斗系统不是并列模块，而是相互支撑的一套整体体验
3. 剧本既承担内容消费，也承担社交关系建立与长线复玩价值
4. 高难战斗、副本门票、资源共享、功法与外观积累，会共同影响社交关系和商业化体验

## 不可丢失的产品特色

1. 双人共斗不是附属玩法，而是核心体验来源
2. 近战位与远程位存在明确分工
3. 合印必须体现两名玩家的配合价值，不能退化成普通双人增伤按钮
4. 剧本、多结局、社交关系与战斗系统是相互支撑的，不应割裂理解
5. 不能把产品误缩成只剩“单角色战斗验证”的普通 ARPG 项目
6. 即使当前阶段在做单角色本地验证，也要明确这只是验证手段，不是产品长期形态

## 对 AI 的要求

1. 设计和实现建议不能破坏双人协同的长期扩展方向
2. 即使当前做单角色本地验证，也要为未来双人共斗与联机扩展预留结构
3. 任何“为了省事改成单人通用 ARPG 方案”的建议都应视为偏离项目方向
4. 若某个局部方案会伤害剧本、社交、双人协作或合印配合价值，应优先指出，而不是只按局部实现成本决策
