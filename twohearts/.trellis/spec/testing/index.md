# 双心印测试规范索引

## 用途

这一层定义双心印当前的测试文档风格、白盒问题记录方式和阶段验收检查口径。

## 开工前必读

1. 涉及跑测、白盒定位、问题记录，先读 [whitebox-testing-guidelines.md](./whitebox-testing-guidelines.md)
2. 涉及测试文档维护边界，先读 [test-document-rules.md](./test-document-rules.md)

## 质量检查

1. 问题记录必须可复现、可修复、可回归
2. 不在测试视角下擅自补设计规则
3. 文档要简洁，但不能丢掉预期/实际/影响范围
