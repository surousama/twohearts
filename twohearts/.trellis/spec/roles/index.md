# 双心印角色职责索引

## 用途

这一层收束四个核心角色的职责边界，避免 AI 在一次任务里同时扮演多个角色、输出方式混乱。

## 开工前必读

1. 先根据任务判定角色
2. 再只读对应角色文档
3. 如当前任务跨角色，先说明当前主角色，再引用其他角色结论，不直接混写
