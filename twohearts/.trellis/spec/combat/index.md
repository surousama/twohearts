# 双心印战斗规范索引

## 用途

这一层收束战斗系统的长期上游规则和当前阶段口径。凡是涉及普通攻击、闪避、格挡、受击、预输入、结印、合印的任务，都应先读这里。

## 开工前必读

1. 涉及长期模块边界、战斗结构、状态和验收边界，先读 [system-framework.md](./system-framework.md)
2. 涉及当前第二章阶段口径、先后顺序和是否适合开工，先读 [current-stage.md](./current-stage.md)

## 质量检查

1. 不要把子功能局部实现成与总框架冲突的孤岛
2. 当前单角色本地验证不代表可以无视未来双人共斗与联机扩展
3. 所有建议都应服从当前阶段顺序
