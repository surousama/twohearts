1. 涉及核心技术路线与正式承载位置，先读 [ue5-gas-cpp-guidelines.md](./ue5-gas-cpp-guidelines.md)

3. 涉及主文档、实施文档、留档文档的更新与收束，先读 [document-maintenance.md](./document-maintenance.md)

## 质量检查

2. 不为了未来泛化需求提前造过重结构
3. 稳定经验要及时回写文档
