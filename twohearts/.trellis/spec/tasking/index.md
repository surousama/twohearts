# 双心印任务系统规范索引

## 用途

这一层定义双心印如何使用 Trellis task，而不是沿用通用“单任务平铺”习惯。

## 开工前必读

1. 先读 [parent-child-task-rules.md](./parent-child-task-rules.md)
2. 再读 [prd-rules.md](./prd-rules.md)

## 质量检查

1. task 粒度采用父 task + 子 task 混合模式
2. 子 task 必须可单轮验收
3. `prd.md` 必须明确“不做什么”
