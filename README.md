# 双心印 (TwoHearts)

## 项目简介
以1v1社交和双人合作玩法为核心的新游类型

## 项目特色
- **创新社交模式**：专注1v1深度社交，打造高粘性用户关系
- **双人合作战斗**：结印+合印系统，重新定义合作体验
- **多结局剧本**：突破传统叙事框架，提供丰富的故事体验
- **资源互补设计**：男女玩家需求互补，形成生态平衡

## 项目结构
- `/docs` - 项目文档
- `/src` - 源代码
- `/assets` - 游戏资源
- `/design` - 设计文档
